VPS
GigsGigsCloud试坑
Jul 21, 2019

前言

最近VPS都在涨价，就连Bandwagon家的最便宜的方案都要50刀/year。虽然之前购买过便宜套餐的用户可以通过续费一直维持下去，但我想趁这个机会试试别家的VPS。GigsGigsCloud家的HK VPS非常便宜，但是网上评价褒贬不一，据说非常不稳定因此被戏称作GG Boom。于是我决定先买个1个月试试水。

套餐选择

GigsGigsCloud的机房在HK和SG，提供CLOUDLET V，CLOUDLET K，CLOUDLET B三种套餐。其中V是高端套餐，电信联通移动直连；CLOUDLET K是KVM架构，据称对特定线路有优化；CLOUDLET B是普通套餐，openVZ架构。我选择的是CLOUDLET K（HK-China），因为V太贵了，数据还很少，而B套餐太便宜了，不敢买。你可以在clientarea查看详细的套餐列表，部分套餐如下图。我买的5.8刀/月的套餐已经没有了，最便宜的是8.8刀/月的套餐，此外可以用网友提供的推荐码打折，但貌似只能打折一次。

我购买的的套餐参数如下：

项目 参数
Price 5.8$/month
CPU 1*vCore
RAM 500M
Storage 15G SSD
IP 1*IPv4
Bandwidth 30Mbps Max
Data Transfer 400GB
Location HK

  1. 带宽
    由于我的水管小，只有10M，所以不怎么好测试。我用speedtest测了一下，可以跑满。

  2. 稳定性
    没遇到过网友说的情况。但GigsGigsCloud的确实给人感觉比较卡，rebuild系统比Bandwagon慢（重置密码花了十几分钟）。
