Hello! Hexo

Hexo写博客

自己一直想写博客，但是由于懒，对网页开发也不感兴趣，更不愿意每年掏钱租服务器，一直都没付诸行动。我经常需要上网查资料，但是有些资料已经是古董级别了，也没人更新，导致走了很多弯路。偶然看到某校友使用 Hexo + Githubpages 搭博客，既不用购买服务器也不需要学习网页开发技术，很是适合我。

Hexo的安装与使用

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Hexo官网提供了详细的安装步骤，还能从官网链接中下载皮肤。
安装好后，配置文件一般保存在_config.yml文件中，Hexo使用起来很简单，只需要记住三个命令。

1. hexo new 创建新的文章

2. hexo g 生成静态文件

3. hexo d 部署博客

使用Markdown

Hexo 使用 Markdown 解析文章，所以还需要学习Markdown的基本语法才能写出漂亮的博客。Markdown远没有高级编程语言复杂，类似于Html，网上有很多教程，我就不贴链接了。值得一提的是我现在使用的编辑器 Atom 可以支持Markdown预览，使用快捷键 Ctrl+Shift+M 就可以预览部署后的页面的样子，对初学者来说非常有帮助。

使用图床

Github Pages拥有300M免费空间，但是如果我们上传图片到Github Pages上，肯定是不够用的。所以需要图床来代替本地图片链接。网上有很多图床并且有免费额度的流量。我采用的是Imgur，能使用Google或者Facebook账号登录。根据Markdown语法，插入图片的代码为

![img](http://i.imgur.com/XXX.jpg)

让别人找到你

博客部署了，但是为什么一直没有访问量呢？（绝望了，对写了博客却没人看而我还在写博客的世界绝望了）

![img](http://i.imgur.com/TOeAzzI.gif)
我使用Baidu，Google，Bing搜索自己文章题目发现根本没被搜索引擎爬取到。可能是各个搜索引擎爬虫对Github Pages的爬取策略的锅。我们需要将自己的博客提交给各个搜索引擎的链接提交平台，提交链接如下。

Baidu

http://zhanzhang.baidu.com/linksubmit/url

Google

https://www.google.com/webmasters/tools/submit-url?hl=zh-CN&pli=1

Bing

https://www.bing.com/toolbox/submit-site-url/

这里仅仅是提交了请求，收不收录还得看大佬们的心情了，我的情况是Google最快收录，Bing第二，而Baidu至今也没收录（我看你这是刁难我胖虎）。
